# React Starter (Vite)

### Included libraries
- react 18.3.1
- react-dom 18.3.1
- vite
- axios
- react-router-dom
- redux
- @reduxjs/toolkit
- react-redux
- @mui/material
- @mui/icons-material
- react-icons
- victory
- lottie-react
- js-cookie

### Quick start
```bash
npm install
npm run dev
```

Notes:
- This project uses Vite to avoid CRA/react-scripts version conflicts.
- If you want TypeScript, I can create a TS version instead.
